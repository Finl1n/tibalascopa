# Referência de Design

## Referência visual

Site usado como inspiração: EYRC Architects, https://www.eyrc.com/

## O que vamos aproveitar como direção

- sensação de sofisticação e calma visual
- composição limpa, com bastante respiro
- tipografia forte e elegante
- navegação discreta e objetiva
- uso de seções bem separadas
- foco em apresentação editorial

## O que não vamos copiar

- identidade visual da marca
- estrutura exata das páginas
- textos, imagens e assets
- soluções proprietárias de navegação

## Tradução para o nosso projeto

Para a Copa Intelligence, essa referência sugere uma interface com:

- layout premium e minimalista
- contraste visual forte entre títulos e conteúdo
- cards bem organizados para estatísticas
- seções em destaque para jogos, jogadores e fatos históricos
- microinterações sutis, sem exagero
- sensação de produto editorial e confiável

## Diretrizes práticas

- evitar aparência genérica de dashboard padrão
- usar uma paleta sóbria com pontos de destaque
- priorizar leitura rápida e clareza dos dados
- manter hierarquia tipográfica bem marcada
- explorar blocos visuais amplos com boa distribuição

## Decisão registrada

Essa referência fica aprovada como direção estética do projeto, apenas como inspiração de qualidade visual e composição.
