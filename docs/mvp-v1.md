# MVP v1

## Objetivo da primeira versão

Entregar uma plataforma funcional com visual premium para consultar informações da copa e interagir com um agente de IA de forma opcional.

## Escopo incluído

### 1. Home dashboard

- cards com destaques gerais
- resumo de jogos
- ranking básico de jogadores
- bloco de fatos históricos
- CTA para abrir o agente de IA

### 2. Jogos

- listar partidas
- exibir placar
- exibir data e status
- destacar principais eventos e estatísticas do jogo

### 3. Jogadores

- listar jogadores relevantes
- exibir métricas principais
- permitir comparação simples

### 4. Histórico

- mostrar fatos históricos de outras copas
- exibir curiosidades e marcos relevantes

### 5. Agente de IA

- aceitar perguntas em linguagem natural
- responder com base em dados internos
- lidar com jogos, jogadores e fatos históricos

## Fora do escopo da primeira versão

- login e autenticação de usuários
- painel administrativo completo
- multi-idioma
- notificações em tempo real
- edição manual avançada de dados pelo usuário
- múltiplas competições simultâneas

## Critérios de sucesso

- o usuário consegue navegar sem depender do chat
- o agente responde com base em dados disponíveis
- a interface transmite sensação premium
- os dados principais aparecem com clareza
- a estrutura permite evolução futura

## Decisão de produto

A primeira versão será focada em leitura, descoberta e interação assistida por IA, sem complexidade desnecessária.
